Timeline Cursor Icons

Visual reference for the custom SVG cursors used by the timeline editing UI.

Source styles live in src/features/timeline/theme.css.
Smart trim logic lives in src/features/timeline/utils/smart-trim-zones.ts.
Cursor Class Preview Meaning Used For
cursor-scissors Razor / cut Razor tool hover
cursor-gauge Rate stretch Rate stretch hover and active drag
cursor-trim-left Left-edge trim / ripple Trim Edit on left edge
cursor-trim-center Center cut / roll Trim Edit at active cut
cursor-trim-right Right-edge trim / ripple Trim Edit on right edge
cursor-slip-smart Slip Trim Edit lower clip body
cursor-slide-smart Slide Trim Edit top label row
Notes
  • V is plain Select again; it uses the normal arrow cursor on clips.
  • T activates the new Trim Edit tool with smart trim/roll/slip/slide behavior.
  • Active drag cursors are applied globally so the cursor does not flicker away during the gesture.
